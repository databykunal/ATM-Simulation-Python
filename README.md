# 💳 ATM Simulation System (Python)

## 📌 Overview

This project is a  ATM simulation built using Python. It allows users to perform basic banking operations like login system, checking balance, depositing money, withdrawing money, and viewing transaction history.

🚀 Features

* PIN Authentication (3 attempts security)
* Balance Enquiry
* Deposit & Withdraw
* Transaction History

 🛠️ Tech Used

* Python
